# 应用缓存处理
	
* 网络加载资源缓存

	```
		网络资源加载 NSURLCache 
		https://github.com/reactnativecn/react-native-http-cache
		来管理
	```
* 网络图片缓存 （见图片说明）
	* 系统默认缓存 https://github.com/reactnativecn/react-native-http-cache 就可以 
		* 图片保存的为URLRequest 请求缓存 而不是Image真实数据
		* RCTHttpCache 是实现很正确
	* 使用第三方图片缓存框架 react-native-img-cache
